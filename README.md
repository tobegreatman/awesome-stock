# 股票实时行情速览系统

一个基于 Vue + Vite + Koa2 + ECharts 的实时行情应用，支持自定义股票池、分时与多周期走势查看，以及行业涨跌幅矩形树图。

## 功能

- 实时股票池：通过代码动态配置，展示名称、涨跌幅、成交额和微型走势。
- 详情走势图：支持分时、日 K、周 K、月 K 切换，并带拖动缩放查看历史数据。
- 市场概览：按行业涨跌幅展示矩形树图，红涨白平绿跌，颜色深浅随幅度变化。
- 后端代理：Koa 统一代理真实行情接口，前端无需处理跨域问题。

## 启动

```bash
npm install
npm run dev
```

开发环境：

- 前端： http://localhost:5173
- 后端： http://localhost:3000

生产构建：

```bash
npm run build
npm start
```

## 数据来源

- 东方财富公开行情接口，用于股票列表、分时、K 线与行业板块数据。

## 环境说明

- 已在当前仓库内加入 `.npmrc`，默认跳过依赖安装脚本，以规避当前 Windows 环境下 `esbuild` 安装脚本异常。
- Node 18+ 可运行。
