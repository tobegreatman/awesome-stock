# 股票实时行情速览系统项目概览

## 1. 项目定位

这是一个基于真实行情数据构建的全栈股票看盘系统，面向“快速盯股票池 + 放大单只走势 + 扫行业结构”三个核心场景。项目将股票池、个股详情和市场概览集中在一个单页应用中，适合本地运行、演示和继续扩展。

核心目标：

- 支持通过股票代码动态维护股票池。
- 支持分时、日 K、周 K、月 K 多周期走势查看。
- 支持行业涨跌幅矩形树图，快速观察市场热度分布。
- 使用后端代理真实接口，降低前端跨域和数据适配复杂度。

## 2. 技术栈

### 前端

- Vue 3：单页应用和组件状态管理。
- Vite：开发服务器和生产构建。
- ECharts 5：分时图、K 线图、成交量图、矩形树图。
- 原生 Fetch：通过前端 API 封装调用后端接口。

### 后端

- Node.js 18+
- Koa 2
- @koa/router
- @koa/cors
- koa-static
- axios：请求东方财富公开行情接口。

## 4. 系统架构

### 4.1 前后端分层

- 前端只调用本地 `/api` 接口，不直接请求第三方行情源。
- 后端统一代理东方财富接口，并把原始返回转换成前端可直接渲染的数据结构。
- 图表配置逻辑集中在 `src/lib/charts.js`，避免组件内部堆积 ECharts 细节。

### 4.2 数据流

```text
用户操作
  -> Vue 组件触发刷新/切换/搜索
  -> src/services/api.js
  -> server/app.js
  -> server/services/eastmoney.js
  -> 东方财富公开行情接口
  -> 后端标准化返回
  -> App.vue 更新状态
  -> 各业务组件渲染卡片、图表和矩形树图
```

### 4.3 状态组织

`src/App.vue` 负责全局状态与主流程控制，主要包括：

- 当前视图：`quotes` / `market`
- 股票池代码列表
- 当前选中的股票代码
- 当前走势周期：`intraday` / `day` / `week` / `month`
- 股票池行情数据
- 单只股票趋势数据
- 行业板块数据
- 各区域加载态、更新时间和错误态

## 5. 核心功能

### 5.1 行情视图

#### 股票池管理

- 支持通过股票代码批量更新股票池。
- 支持多种分隔方式：空格、逗号、分号、换行等。
- 支持股票搜索建议与自动补全。
- 股票池会同步到 URL 参数和本地存储，刷新页面后可恢复。

#### 股票池列表

- 展示股票名称、代码、最新价、涨跌额、涨跌幅、成交量、成交额。
- 每只股票带最近走势微图，便于快速比对强弱。
- 当前列表高度已限制为约 4 个卡片，超出后内部滚动查看。

#### 个股详情

- 支持分时、日 K、周 K、月 K 四种周期切换。
- 分时图展示价格线与均价线。
- K 线图展示蜡烛图与成交量。
- K 线支持拖动和滑块缩放查看更早数据。
- 日 K、周 K、月 K 默认显示窗口已按周期优化，打开时更容易观察均线关系。

#### 指标与观察模式

- 支持多组指标卡片展示关键数值。
- 支持 MA5、MA10、MA20、MA60 均线组合切换。
- 详情区部分偏好支持本地持久化，例如指标卡片模式、均线开关。
- 提供恢复默认观察模式的入口，便于快速回到标准视图。

### 5.2 市场概览视图

- 使用行业板块矩形树图展示市场热度结构。
- 颜色遵循红涨、白平、绿跌，且颜色深浅随涨跌幅变化。
- 支持按方向过滤：全部、上涨、平盘、下跌。
- 支持按排序维度切换：涨幅、资金、强度。
- 支持按面积权重切换：综合、资金、涨幅。
- 提供空状态提示和一键恢复默认控制入口。

### 5.3 顶部摘要与导航

- 支持在“行情”和“市场概览”之间切换。
- 支持手动刷新当前视图所需数据。
- 顶部摘要区域会展示股票池平均涨跌、最强标的、行业领涨等信息。
- 当前活跃视图和部分市场控制项支持持久化。

## 6. 后端接口

后端入口文件为 `server/app.js`，统一挂载在 `/api` 前缀下。

| 接口 | 说明 |
| --- | --- |
| `/api/health` | 健康检查 |
| `/api/quotes?codes=...` | 获取股票池实时报价和微图数据 |
| `/api/search/stocks?query=...` | 股票名称/代码搜索建议 |
| `/api/trend/:code?period=...` | 获取分时或 K 线走势 |
| `/api/market/industries?limit=...` | 获取行业板块涨跌和资金流样本 |

`server/services/eastmoney.js` 负责：

- 请求东方财富行情接口
- 代码与市场 ID 转换
- 分时/K 线字段解析
- 行业板块数据合并与排序
- 搜索建议中文名称适配
- 短时缓存，降低频繁刷新压力

## 8. 运行方式

### 开发环境

```bash
npm install
npm run dev
```

说明：

- `npm run dev` 实际执行 `node scripts/dev.js`
- 默认会同时拉起：
  - 前端：`http://localhost:5173`
  - 后端：`http://localhost:3000`

### 生产构建

```bash
npm run build
npm start
```

说明：

- `npm run build` 使用 Vite 输出到 `dist/`
- `npm start` 启动 Koa 服务，并在生产环境下托管静态资源
